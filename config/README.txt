Requirements:
	php 5 or higher must be installed
	php_pdo_sqlite.dll must be enabled in php.ini

	(project built and tested on windows 7)


Run:
	navigate to blog\bin folder in command line
	run command: "cake server"
	in a browser go to localhost:8765

Explore all features:
	several accounts have been premade, one for each type of account and the default Guest user that does not require a login.
	account names are: "admin", "author" and "user"
	each account password is the same as its username
	logging into each account will allow you to explore the website as each user type.
